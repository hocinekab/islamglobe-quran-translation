# 📖 IslamGlobe: Quran Translation

A Quran reading application designed to improve focus and respect the visual importance of the Quranic text.

## Concept

This project rethinks the visual hierarchy of Quran reading on digital screens:
- The Quranic text is visually dominant
- Translation is subtle and secondary
- Reading is distraction-free (no ads)

## Features

- Clean and fluid reading experience
- Translation displayed as a support, not a distraction
- Tap on verse number to access explanation/definition
- Focused UI for immersive reading

## Goal

The goal is not to create a "better Quran", but to improve how it is experienced in the digital age.

## Status

First version released. Continuous improvements in progress
